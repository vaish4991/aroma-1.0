Enable real authentication in 2 minutes:

  1. Go to console.firebase.google.com
  2. Create project → Add Web App → Copy config
  3. Open firebase-config.js and paste your keys
  4. Enable Email/Password, Google & GitHub in Authentication
  5. Refresh this page
